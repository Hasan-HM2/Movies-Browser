# 🎬 Movies Browser

A modern, fully responsive movie browsing web application built with **React**, **Material-UI**, and integrated with the **TMDB API**.

## ✨ Features

- **Dynamic Search:** Real-time movie filtering using a seamless, responsive search bar.
- **Personalized Watchlist:** Add your favorite movies to your local watchlist, saved persistently using browser `localStorage`.
- **Cinematic UI/UX:** Stunning detail pages with blurred backdrop filters and elegant transitions.
- **Fully Responsive:** Optimized experience from small mobile screens up to desktop displays.
- **Clean Routing:** Built with React Router, featuring smooth transitions and an integrated 404 Error page.

## 🛠️ Built With

- **Frontend:** React.js, React Router
- **Styling:** Material-UI (MUI), CSS
- **API:** The Movie Database (TMDB) API
- **State Management:** React Context API
